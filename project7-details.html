Terraform AWS Modular Infrastructure

  • Designed and implemented a modular Terraform architecture to provision AWS infrastructure with reusable, maintainable code.
  • Created a custom VPC with private subnets for EC2 instances and public subnets for the ALB.
  • Launched EC2 instances in private subnets, attaching appropriate IAM roles and instance profiles for secure AWS access.
  • Configured an Application Load Balancer (ALB) in the public subnet to distribute traffic and ensure high availability.
  • Used S3 for static content storage and configured as a remote backend for Terraform state management.
  • Enabled state locking with DynamoDB to ensure safe concurrent infrastructure changes.
  • Deployed a web application on EC2, accessible through the ALB DNS.
  • Followed best practices for modular architecture, improving reusability, clarity, and version control of infrastructure components.
